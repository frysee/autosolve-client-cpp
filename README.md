# AYCD AutoSolve C++ Client

Client for connecting to the AYCD AutoSolve network

## Dependencies

- **Qt** >= 6.0 (tested w/ 6.2.3)
- **[qamqp](https://github.com/frysee/qamqp)** >= 0.6.2
